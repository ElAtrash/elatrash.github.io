# Safiya El Atrash - Developer Portfolio

This is my developer-focused CV landing page hosted on GitHub Pages.

## Features

- Minimalist, developer-friendly design
- Responsive layout that works on all devices
- Clean typography using Inter and Fira Code fonts
- Organized sections for professional summary, skills, experience, education, and languages
- Timeline-based work experience display
- Skills organized in a grid layout
- Language proficiency indicators

## Sections

- Professional Summary
- Skills (categorized in a grid layout)
- Work Experience (with timeline)
- Education
- Languages (with proficiency indicators)

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Inter and Fira Code)
